## Auto-Update Setup

The dashboard auto-updates 4× daily via GitHub Actions + Fantrax API.

### Setup Steps

1. **Fork or create this repo** on GitHub (make it **Public** for free GitHub Pages)

2. **Upload all files** maintaining this structure:
   ```
   /
   ├── index.html              # Main dashboard
   ├── fetch_data.py           # Fantrax data fetcher
   ├── README.md               # This file
   ├── .github/
   │   └── workflows/
   │       └── update-data.yml # GitHub Actions workflow
   └── data/
       └── league_data.json    # Auto-updated league data (seed file included)
   ```

3. **Add FANTRAX_SESSION secret**:
   - Open Chrome, go to fantrax.com and log in
   - Press F12 → Application tab → Cookies → fantrax.com
   - Find the cookie named **`session`** and copy its value
   - Go to your GitHub repo → Settings → Secrets and variables → Actions
   - Click "New repository secret"
   - Name: `FANTRAX_SESSION`, Value: (paste the cookie value)

4. **Enable GitHub Pages**:
   - Go to repo Settings → Pages
   - Source: Deploy from branch → **main** → **/ (root)**
   - Click Save
   - Site will be live at: `https://YOUR_USERNAME.github.io/REPO_NAME/`

5. **Run initial data fetch**:
   - Go to Actions tab → "Update League Data" → "Run workflow"

6. **Share the URL** with all league members!
